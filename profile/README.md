**Installation Guide**  
1. Download the latest trainer archive.  
2. Extract to a dedicated folder outside Steam directories.  
3. Launch NARAKA BLADEPOINT via Steam.  
4. Run NBTrainer.exe as administrator.  
5. Press Insert in-game to open the trainer menu.  
6. Activate features and verify god mode or currency in the first match.  

**Safety & Risk Information**  
External memory trainer with dynamic pattern scanning. High detection risk on official servers and public matches. Aggressive ESP or god mode may trigger anti-cheat or bans. Extreme modifications can cause temporary desync. Use primarily in private lobbies. Restart trainer after major patches. Possible minor visual artifacts during intense combat.
